# cursor-boston-hackathon

## What we’re building (ACL only)

An ACL-focused triage + guidance tool that takes:
- **Text intake** (mechanism, pop, swelling, instability, etc.)
- **Short user-selected video clip** (5–15 seconds, ideally cropped to the injured athlete)

…and returns:
- **ACL concern band** (low / moderate / high / insufficient evidence)
- **Urgency** (self-care / see clinician soon / ortho or sports med soon / emergency now)
- **Confidence + rationale + next steps**

## Run frontend

```bash
cd frontend
npm install
npm run dev
```

Open: `http://localhost:3000`
